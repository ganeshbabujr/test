1. NUMBER

NUMBER(P,S)         0-9        i.e employee-id NUMBER         -- P- Precision - no. of digits in a number
Size: Optional                                              -- S Scale - no of digits in Decimal point
Max size: 9999...9(38)

i.e 
   salary_pt       569.45
   salary        NUMBER(5,2)            -- 5 is Total Digit in Number and 2 is decimal
   
i.e 
   salary_pt      23987.985
   salary       NUMBER(8,3)

2. VARCHAR

Variable Character LENGTH --oracle changed VARCHAR to VARCHAR2 if u use VARCHAR it change to VARCHAR2 internally

VARCHAR2        0-9 | A-Z | a-z | Special Character        i.e first_name VARCHAR2(30) | email VARCHAR2(40)
Size: Mandatory
Max: 4000            --Upto 11g we can use Max 4000 In 12C|18c|19c we can Increase the Size i.e 7000 up to 32767
ALTER SYSTEM SET max_string_size = EXTENDED     --to increase the max size upto 32767
ALTER SYSTEM SET max_string_size=DEFAULT        -- to set default size
ALTER SYSTEM SET nls_date_format=DD/Mon/YYYY    -- To change the Date Format of Oracle from DD Mon YY to any i.e DD/Mon/YYYY
-- DBA team will use this query

Diff between VARCHAR and VARCHAR2

VARCHAR is a Data Type which is replaced by VARCHAR2 now.

Inserting Data with Sub Query in 'INSERT'

-- Creating table without DATA
CREATE TABLE ganesh_tb
AS 
(SELECT * FROM departments);

INSERT INTO ganesh_tb (SELECT * FROM departments);

UPDATE

-- Changing the name department_name Administrator as Admin  
UPDATE ganesh_tb
   SET department_name='Admin'
WHERE department_id=10;

--Changing 2 or more data in same row.
UPDATE ganesh_tb
   SET department_name='Admin'
   ,   manager_id=101
   ,   location_id=1500
WHERE department_id=10;                                         

-- If u Miss the WHERE clause it will update for all rows
UPDATE ganesh_tb                                                                  --ERROR
   SET department_name='Admin'
   ,   manager_id=101
   ,   location_id=1500

DELETE

DELETE FROM ganesh_tb WHERE department_id=10;

--Delete in Multiple 
DELETE FROM ganesh_tb WHERE department_id IN (10,40,60,80,100);

-- If u dont give condition it will delete entire Data like TRUNCATE but there is Difference
DELETE FROM ganesh_tb;

Diff Between Truncate and Delete

TRUNCATE is a DDL Statement while DELETE is a DML Statement
